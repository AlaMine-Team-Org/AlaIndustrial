## 0.1.29

New storage tier, a mute chip for any machine, more ways to feed lava into the geothermal generator, and a fix for a flicker on chest lids.

### New

- **Silver Chest** — 45 slots, the tier above iron. Full 3D chest with animated lid. Crafts from an iron chest + 8 silver ingots. No power needed, any pickaxe breaks it.
- **Mute Chip** — put it in a machine's upgrade slot to silence that block completely. Pull it out, sound comes back. Drops when the machine is broken.
- **Geothermal lava input** — lava capsules now work just like lava buckets: drop them in the input slot, or shift + right-click the block to fill the tank straight from a bucket. Empty container always comes back.
- **Lava capsule as furnace fuel** — same burn time as a lava bucket. Returns an empty capsule.

### Quality of Life

- **Upgrade panel** — a gear button on every machine and generator opens a 4-slot panel inside the same window. Draggable (position saved), closes with an X, and a mute chip slots in via shift + left-click.

### Fixes

- **Flicker along the chest lid seam** — iron and silver chests no longer flicker where the lid meets the base.
