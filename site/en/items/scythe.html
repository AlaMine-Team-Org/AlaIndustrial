Scythe

alaindustrial:scythe
ItemsImplementedtoolscytheaoefoliage

Summary

The Scythe is an Ala Industrial hand tool for quickly clearing decorative foliage. A right-click on a block mows down every eligible plant (leaves, grass, flowers, ferns, mushrooms, vines, roots, saplings) in a flat rectangular area in front of the player, dropping normal vanilla loot exactly as if the player had broken each block by hand. Eight material tiers (wood → stone → copper → iron → gold → tempered iron → diamond → netherite) scale the area and the per-swing block cap. The scythe is a quality-of-life tool for building and gathering decorations; it does not replace the axe, the hoe or an auto-farm: the first version deliberately does not harvest #minecraft:crops.

Energy

An ordinary hand tool — it does not connect to the mod's energy network and uses no EU/FE.

FieldValue
Rolenone (hand tool, not part of the EU/FE system)

Inventory

An item. Stacks to 1 (a durability tool). It carries the minecraft:tool data component via Item.Properties.hoe(material, -2.0f, -1.0f), but it is not a HoeItem — the scythe does not till dirt on right-click; it runs its own AOE. Repaired with the matching tier material (vanilla repair tags; tempered iron via #alaindustrial:tempered_iron_tool_materials).

Not applicable (an item).

Stats (balance)

The area is a flat width × depth × height box in front of the player: depth runs forward along the look direction, width (always odd — there is a centre line) along the perpendicular horizontal axis, height is always 3 (the clicked layer ±1). The centre is the clicked block. Durability and mining speed come from the vanilla ToolMaterial; the numbers live in code (/ModItemsNeoForge via ScytheItem.Profile), not in `C.

Gold is a deliberate side-grade: iron-sized area, but fragile (32 durability) and highly enchantable (22) — the vanilla gold philosophy (shines with Unbreaking/Mending). Copper is a modest step between stone and iron. The netherite scythe is fire-resistant (.fireResistant), like all vanilla netherite gear. The attack profile mirrors a hoe (hoe(material, -2.0f, -1.0f)): low damage, not intended as a weapon.

Target blocks

Only blocks in the alaindustrial:scythe_harvestable tag (data/alaindustrial/tags/block/scythe_harvestable.json) are mowed: sub-tags: #minecraft:leaves, #minecraft:flowers (includes #small_flowers), #minecraft:saplings, #minecraft:cave_vines; grass/ferns: short_grass, tall_grass, fern, large_fern, dead_bush, bush, firefly_bush, short_dry_grass, tall_dry_grass, leaf_litter; other: moss_carpet, pale_moss_carpet, brown_mushroom, red_mushroom, vine, glow_lichen, hanging_roots, nether_sprouts, warped_roots, crimson_roots. Excluded (deliberately for v1): #minecraft:crops (a separate farming mechanic), pumpkin/melon stems, sugar cane, cactus, bamboo, kelp (farming/resources), logs, dirt, moss block (terrain), and water/seagrass. chorus_flower comes in transitively via #minecraft:flowers — accepted as a decorative block.

Behaviour

Trigger: only a right-click useOn, only from the main hand. Shift/secondary use and the off-hand return PASS (normal interaction is not blocked). Breaking: via ServerPlayerGameMode.destroyBlock(pos) — the canonical "player mined a block" path. That gives, for free: tool-aware loot (Fortune/Silk Touch work), correct double-plant handling (the other half disappears without a duplicate drop), automatic durability, creative parity (no drop, no durability), adventure/spawn-protection, and both loaders' block-break events (Fabric PlayerBlockBreakEvents / NeoForge BlockEvent.BreakEvent) firing — protection mods can veto each block. Durability (vanilla tool rules): 0-hardness blocks (short grass, fern, flowers) break instantly and cost no durability; blocks with hardness > 0 (leaves at 0.2) cost 1 durability each. Grass can be mowed forever; leaves cost durability per the tier. Cap: never more than the tier's max blocks/swing; if the tool breaks mid-swing the pass stops (it never breaks past the available durability). Candidates are sorted by distance from the click — the clicked block breaks first, so cap and durability are deterministic. Sound: one swing/cut alaindustrial:scythe_swing per successful swing (not per block), source PLAYERS, with a slight pitch variance. Effect: a minecraft:sweep_attack particle flourish over the cleared area, spread across the scythe's width/depth (more when more is cleared).

Progression

Tier: none (a hand tool, not tied to the mod's energy tiers). Eight material steps parallel the vanilla hoes; in creative each scythe sits right after its tier's hoe (the Tools & Utilities tab). Area and cap grow with tier; gold is a fragile enchantable side-grade, tempered iron a step between iron and diamond.

Compatibility

Recipe viewers: JEI, REI, EMI — as ordinary crafted items. Enchantments: the scythe is in #minecraft:hoes, so via the #minecraft:enchantable/* chain it accepts efficiency/unbreaking/fortune/silk_touch/mending. Fortune boosts the drop (the break path is tool-aware); Efficiency does not affect the instant AOE. Protection/adventure: the scythe does not bypass vanilla restrictions or the loaders' break events.
